第一题

关键词: 思维题 Codeforces

题目链接

题目描述

有n个选手，参加了5场比赛，已知5场比赛中每位选手的排名。
在五场比赛中，如果至少有3场比赛中$a$的排名高于$b$，则认为选手$a$比选手$b$强。如果一名选手比其它所有选手都强，则认为他“很可能得金牌”。请找出“很可能得金牌”的选手。如是不存在输出$-1$,如果存在多个输出任意一个即可。

题解

本题中选手的相对强弱关系不具有传递性($a$强于$b$, $b$强于$c$, 不能推出$a$强于$c$)，不能简单地对选手进行排序。
对于任意两名选手$a$和$b$,要么$a$比$b$强，要么$b$比$a$强，不存在“$a,b$不能判断强弱”的情况（因为是五局三胜）。“很可能得金牌”的选手比其它所有选手都强。任意比较两名选手$a$和$b$，如果$b$比$a$弱，则$b$一定不是“很可能得金牌”的选手。于是，我们可以先比较选手$1$,$2$, 排除掉弱的选手，剩下的一名选手再与$3$比较，再排除掉弱的选手，…依此类推， 最后只剩下一位选手有可能上”很可能得金牌”的选手。但他并不一定是”很可能得金牌”的选手, 因此还要将他与其它所有选手进行比较，如果他比其它所有选手都强，则他是”很可能得金牌”的选手，否则不存在”很可能得金牌”的选手。时间复杂度$O(n)$

第二题

关键词: 思维题 Codeforces

题目链接
瞎写了一堆代码，比赛结束前16秒，样例没测直接提交，莫名其妙地过了

题目描述

一只蚂蚁在数轴上，从$x = 0$开始向正方向移动。数轴上有$n$个传送门，第$i$个传送门位于$x_i$，可将蚂蚁传送至$y_i (y_i < x_i)$,有些传送门都是激活的，有些传传送门是不激活的。

  • 蚂蚁走过一个不激活的传送门，该传送门被激活，蚂蚁继续向前走（不会被传送）。
  • 蚂蚁走过一个激活的传送门，该传送门变为不激活，同时蚂蚁被传送至$y_i$,然后继续向前走。

$n$个传送门满足$x_1 < x_2 < … < x_n$且$x_1, x_2, …, x_n, y_1, y_2, …, y_n$ 这$2n$个整数互不相同。
问蚂蚁从$x=0$开始，走到$x=x_n + 1$需要经过的总路程是多少？

题解

脑补一下蚂蚁的移动过程，不难发现，当蚂蚁走到一个位置时，他背后的所有传送门，都一定是激活的： 如果一个传送门初始是不激活的，蚂蚁走过它把它激活了。如果初始是激活的，蚂蚁走过它把它取消激活，然后蚂蚁被传回去，蚂蚁必须再次走过这个传送门，把它激活。
用一个数组$S_i$表示蚂蚁被第$i$个传送门传回去，再次走到这个传送门所需的时间。可以推出以下递推关系:

  • 第一个传送门: $S_1 = x_1 - y_1$
  • 之后的传送门: $S_i = x_i - y_i + \sum_kS_k$, $k$为$y_i$到$x_i$之间线段上的传送门，即$y_i < x_k < x_i$。

递推求出$S$数组即可，朴素做法时间复杂度为$O(n^2)$，可以一边递推一边维护$S$数组的前缀和$SS$, 每次用二分查找找到$y_i$后的第一个传送门，将$SS_{i - 1} - SS_{t-1}$加到$S_i$即可，时间复杂度优化到$O(nlogn)$。
最后统计答案， 蚂蚁直接走到终点的贡献为$x_n + 1$，途中每个激活的传送门会让蚂蚁多走$S_i$。
